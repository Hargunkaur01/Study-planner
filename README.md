# 📚 Study Planner

A simple and clean study planner web app built using HTML, CSS, and JavaScript.

## ✨ Features
- Add tasks
- Mark tasks as completed
- Delete tasks
- Clean UI design

## 🛠 Tech Used
- HTML
- CSS
- JavaScript
  
 ## 📌 Future Improvements
